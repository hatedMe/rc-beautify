## rc-beautify
> This extension wraps prettydiff to format your javascript, JSX, typescript, TSX file.

## How To Use?
- shortcuts: Ctrl+B
- Press F1, enter Beautify React Javascript TypeScript

## Setting 
changge a global settings:
```javascript
{
    "insize": 4,
    "inchar": " ",
    "bracepadding": false,
    "wrap": 80
}
 ```
